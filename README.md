### Instructions

To compile and deploy contracts:

```sh
truffle compile
truffle migrate
```

To start frontend

```sh
npm run frontend
```

To start backend (hot reload)
```sh
npm run server-dev
```

Type-Check, Compile and Minify for Production

```sh
npm run build
```

Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
